Phrasing styles

Use <em> for stress emphasis and <strong> for strong importance. <i> is now used as an alternate voice and <b> is used for text that is stylistically offset, like drop caps. <sub> is for subscript text, like O2 while <sup> is for superscript text, like this[1].

Code blocks (cb) are styled like below. The class .cb is added to the <pre> tag. A syntax can be displayed in the top right corner of the code block on hover. These can be added by declaring a data-syntax. Additionally a caption can be added, it will be displayed below the code. This caption can be set by declaring a data-caption.


<!-- This is a comment -->
<pre class="cb" data-syntax="HTML" data-caption="This is an HTML code block">
    <code>
	...
    </code>
</pre>
